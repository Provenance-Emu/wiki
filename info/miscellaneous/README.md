# Miscellaneous

This section covers specialized topics that don't fit neatly elsewhere — from running macOS in a virtual machine (to build Provenance without a physical Mac) to launching games via URL scheme and recovering lost save files.

## 💻 Virtualizing macOS

**[Virtualizing macOS](virtualizing-macos.md)** — Don't have a Mac? You can run macOS in a virtual machine on Windows (using VMware) to build or sideload Provenance. Covers setup, configuration, USB device detection, and troubleshooting common VM issues.

## 🔗 Launch ROMs via URL

**[Launch ROMs via URL](launch-roms-via-url.md)** — Provenance registers a custom URL scheme (`provenance://`) that lets you launch specific games from other apps, shortcuts, or home screen icons. Supports launching by MD5 hash, game title, or title + system ID.

## ♻️ Restoring Files

**[Restoring Files](restoring-files.md)** — How to export and restore ROMs, BIOS files, save states, and battery saves — essential when migrating between app versions, changing your Bundle ID, or recovering from a clean install. Covers both Web Server (WebDAV) and Apple Files methods.
